# Medical Record Management System

This is the initial backend setup for a Medical Record Management System (MRMS).

## Tech Stack

- Node.js
- Express.js
- JSON files for data storage

## Project Structure

- `frontend/` is intentionally empty. UI/UX work will be added later.
- `backend/` contains controllers, routes, middleware, services, utilities, and config.
- `data/` contains JSON files used as the starter database.
- `docs/` contains Swagger setup.

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

The server runs at:

```text
http://localhost:3000
```

Swagger documentation is available at:

```text
http://localhost:3000/api-docs
```
